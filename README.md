Student Records Management System (CLI)

A simple Student Records Management System written in C, uses a binary file as storing method.

# FEATURES 

- Add Students
- Remove Student
- Search For Student
- Display Current Students' Data

# HOW TO COMPILE AND RUN 

You can run the .exe file or you can compile the .c file and run it.
Using GCC compiler: 
```bash
gcc main.c -o student_management; ./student_management
```
