Student Records Management System (CLI)

A simple Student Records Management System written in C, uses binary files as storing method.

# FEATURES #

- Add Students
- Remove Students
- Search For Student
- Display Current Students' Data

# HOW TO COMPILE AND RUN #

Using GCC compiler: 
```bash
gcc main.c -o student_management; ./student_management
```
